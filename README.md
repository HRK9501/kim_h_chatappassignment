# Chat App

This project is building basic chatting app using vue, Socket.io and node.


### Dependencies

* HTML5
* CSS3
* SASS/CSS
* JS
* VUE.js
* Node
* Socket.io

### Installing

* cd to the directory of your choice
* git clone the repository link
* Edit and assess!
